About This Dataset

The Cooperative Election Study (CCES) is a large-scale national survey conducted annually. This cumulative file combines data from 2006-2024, standardizing variable names and coding across years for longitudinal analysis.

Custom Functions for Analysis

The following custom functions are available for recoding variables:

  • cces_trad(df, var) - Recode into detailed religious traditions
  • cces_attend() - Recode attendance categories
  • cces_pid3() / cces_pid7() - Recode party ID
  • cces_race() - Recode race categories
  • cces_educ() - Recode education levels
  • cces_inc() - Recode income brackets

Example usage with weighting:

cces %>% group_by(pid3) %>% mean_ci(outcome, wt = weight)

Important Notes

Weighting

Always use the weight variable (commonweight) for population estimates. This weight adjusts for sampling design and non-response.

Missing Data

  • Income: Code 97 = "Prefer not to say"
  • Religious denominations: Variables are NA when not applicable (e.g., baptist is NA for non-Protestants)
  • Family variables: havekids and numkids only available in recent survey years
  • Denomination codes: Most denomination variables use 99 for "Other" or "Something else"

Sample Sizes by Year

To see weighted sample sizes by year, run:

cces %>% count(year, wt = weight)

Religious Tradition Coding

The religion variable follows Pew Research Center categories. For more detailed religious tradition recoding (e.g., Evangelical Protestant, Mainline Protestant, Black Protestant), use the cces_trad() function which combines multiple variables.
